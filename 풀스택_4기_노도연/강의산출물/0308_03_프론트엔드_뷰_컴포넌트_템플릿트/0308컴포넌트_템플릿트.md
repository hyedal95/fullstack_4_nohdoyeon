# 컴포넌트 & 템플릿트    
 -  <!-- 컴포넌트 & 템플릿트 -->

 -  모두 소문자로 사용한다. 
 -   1. 돔요소(DOM) => 케밥-CASE (소문자 사용)
 -      <header-component></header-component>
 -      ↓ 약자로 아래와 같이 쓸 수 있다.
 -      <header-comp></header-comp>
 -  
 -   2. script => 파스칼 케이스 
 -      <HeaderComponent></HeaderComponent>

 -   *프롭스(Props)
 -      부모요소가 자식요소에게 데이터를 전달하는 통신 
 -      단방향 통신 : 부모 => 자식
 -
